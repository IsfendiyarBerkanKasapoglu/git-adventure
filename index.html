Git Quiz

What is Git and why is it so important?

Git is the most commonly used version control system. Git tracks the changes you make to files, so you have a record of what has been done, and you can revert to specific versions should you ever need to.

What’s a branch and why do we create them?

A branch represents an independent line of development. Branches serve as an abstraction for the edit/stage/commit process.

What’s the working tree?

The working tree, or working directory, consists of files that you are currently working on.

What’s the staging area?

The staging area is like a rough draft space, it's where you can git add the version of a file or multiple files that you want to save in your next commit (in other words in the next version of your project).

What’s the command to retrieve the latest version of your files from the remote serve?

git pull
